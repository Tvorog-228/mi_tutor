# 🗣️ AI Language Coach (Experimental)

This is an experimental project developed to explore the combined power of the **Groq API** and native browser capabilities (**Web Speech API**). It is a comprehensive platform designed to master **English or Spanish** through immersive interaction.

## 💡 Why this experiment?

The market is flooded with language learning apps (like Speak, ELSA, or Loora), but almost all of them require expensive monthly subscriptions. 

This project was born out of a single question: **How far can we go using only free and native tools?**
- **Zero Server Costs:** Speech processing (STT and TTS) is delegated entirely to the user's browser.
- **High-Level Inference:** Responses are generated by **Groq**, leveraging its ultra-fast free tier for developers.
- **Total Privacy:** Your data stays in your browser.

## 🚀 Try the App

You can access the functional version directly from your browser (Mobile or PC) here:

> **Important Note:** To use the tutor, you must provide your own **Groq API Key**. The key is stored securely and locally in your browser (`localStorage`) and never leaves your device except to communicate directly with Groq's official servers.

## 🌟 Key Features

- **Multimodal Learning:** Switch seamlessly between **Voice Chat** and **Text Messaging** with the AI.
- **Native Speech Processing (STT):** Uses the browser's `SpeechRecognition` to convert your speech to text without paid external APIs.
- **Smart Voice Synthesis (TTS):** Implements `SpeechSynthesis` with a system voice selector and real-time speed control.
- **Real-World Library:** Practice your reading skills with **authentic articles** directly within the app.
- **Massive Flashcards:** Study thousands of unique words extracted from real books. Includes **integrated AI** to provide instant definitions and usage examples for every card.
- **Dynamic Scenarios:** Configurable modes (Job Interview, Casual Friend, Travel) that adapt the AI's personality and vocabulary.

## 🛠️ Tech Stack

- **HTML5 / JavaScript (Vanilla):** No heavy frameworks (React/Vue) to ensure lightning-fast loading speeds.
- **Tailwind CSS:** Modern, minimalist, and fully responsive mobile-first UI.
- **Web Speech API:** Native browser technology for free, high-quality speech recognition and synthesis.
- **Groq Cloud API:** Sub-second latency AI inference using the `llama-3.3-70b-versatile` model.

## 🧪 Experimental Notes

This project proves that a high-quality language tutor doesn't have to stay behind a paywall. By combining the user's local hardware (the browser) with efficient inference APIs like Groq, we can democratize access to powerful educational tools. **Everything here is 100% free.**
